# StayNest

StayNest is a Node.js and Express web application for browsing and managing vacation stays.

## Features
- Browse listings
- Add and edit listings
- Leave reviews
- User authentication and flash messages

## Tech Stack
- Node.js
- Express
- MongoDB with Mongoose
- EJS
- Passport.js

## Getting Started
1. Install dependencies:
   npm install
2. Start the application:
   npm start
